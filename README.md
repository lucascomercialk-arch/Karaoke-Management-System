#  Karaoke Management System

Sistema desenvolvido para gerenciamento e organização de filas de participantes em eventos de karaokê.

O objetivo é substituir controles manuais e planilhas por uma solução simples, organizada e intuitiva para operadores e participantes.

##  Tecnologias Utilizadas

- HTML5
- CSS3

##  Funcionalidades Atuais

-  Cadastro de músicas
-  Cadastro de participantes
-  Organização automática da fila
-  Controle da ordem das apresentações
-  Interface voltada para operadores do evento
-  Estrutura preparada para exibição pública da fila

##  Objetivos do Projeto

- Melhorar a organização de eventos de karaokê
- Reduzir erros na gestão da fila
- Facilitar o trabalho dos operadores
- Melhorar a experiência dos participantes
